# Deep Research Workflow

## Trigger
Research, deep dive, investigate thoroughly, comprehensive analysis.

## Workflow

1. **Understand** — identify scope, domain, and depth needed.
2. **Decompose** — break the topic into 3-7 sub-questions.
3. **Research** — use web_search with multiple queries per sub-question.
4. **Synthesize** — combine findings into a coherent report with sources.
5. **Save** — store the report in the knowledge base if the user expects persistence.

## Guidelines
- Cite sources explicitly.
- Note confidence levels and conflicting information.
- Identify knowledge gaps for follow-up.
