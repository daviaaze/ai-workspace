# MÓDULO 1 — LOGÍSTICA FÍSICA

## 1.1 O Ciclo Físico do Lote

```
SLE/PF/PRF (depósito) → Seu apto (triagem) → Correios/ML (envio) → Cliente
        |                      |                      |
    Retirada               Teste + foto           Postagem
    (sábado)              (noite/fds)            (próximo dia útil)
```

## 1.2 Retirada do Lote

### Onde retirar
Cada edital indica o depósito fiscal onde a mercadoria está. Locais comuns:

| Órgão | Depósito típico | Horário | Documentos |
|-------|----------------|---------|------------|
| **Receita Federal SLE** | Centro de Atendimento ou Alfândega (portos/aeroportos) | Seg-Sex 9h-17h | Guia de arrematação, DARF pago, CNPJ |
| **PF** | Depósito da SR/PF na capital | Agendado | OFício de arrematação, documento |
| **PRF** | Pátio da PRF na capital | Agendado | Documento do veículo, certidão |
| **Detran** | Pátio conveniado (guincho) | Agendado | Alvará judicial, documento |

### Checklist de retirada

```
📋 ANTES DE IR:
  [ ] DARF pago (comprovante impresso)
  [ ] Documentos: RG, CPF, CNPJ, comprovante arrematação
  [ ] Veículo adequado (carro próprio, alugado, ou fretado)
  [ ] Caixas, plástico bolha, fita adesiva, carrinho de carga
  [ ] Ferramentas: chave de fenda, alicate (pra abrir caixas)
  [ ] Lanterna (depósitos têm pouca luz)
  [ ] Celular carregado (pra tirar foto na hora)

DURANTE:
  [ ] Conferir quantidade vs. edital (contar TUDO)
  [ ] Testar se possível (ligar celular, verificar tela)
  [ ] Fotografar cada item individualmente
  [ ] Se algo veio quebrado: registrar em ata
  [ ] Levar planilha impressa com os itens do lote

DEPOIS:
  [ ] Lavar/limpar itens (álcool isopropílico em eletrônicos)
  [ ] Separar por estado: OK / Com defeito / Para peças
  [ ] Atualizar inventário no sistema
```

# MÓDULO 2 — METODOLOGIA DE ESCOLHA DE LOTES

## 2.1 Critérios de Pontuação (Lote Score)

Cada lote recebe uma nota de 0 a 100 baseada em 7 critérios:

| Critério | Peso | O que mede |
|----------|------|-----------|
| **ROI líquido estimado** | 30% | (Receita líquida - custo total) / custo total |
| **Confiança da descrição** | 15% | Quão detalhado é o lote |
| **Liquidez (velocidade de venda)** | 15% | Dias estimados pra vender tudo |
| **Espaço físico necessário** | 10% | Cabe no apto? |
| **Distância de retirada** | 10% | Até 600km → OK; >600km → penaliza |
| **Familiaridade com o produto** | 10% | Já conhece o mercado desse item? |
| **Sazonalidade** | 10% | Momento certo pra vender (Black Friday, Natal, etc.) |

### Fórmula de pontuação

```python
def lote_score(lote: Lot) -> float:
    """
    Retorna nota 0-100.
    Lotes com score >= 70 → 🟢 RECOMENDADO
    Lotes com score 50-69 → 🟡 ANALISAR
    Lotes com score < 50  → 🔴 PASSAR
    """
    s_roi = min(lote.roi_liquido / 0.5, 1.0) * 30           # ROI >= 50% = nota máxima
    s_conf = lote.confidence_score * 15                       # Confiança
    s_liq = max(0, 1 - (lote.meses_para_vender / 3)) * 15    # Até 3 meses
    s_espaco = espaco_score(lote) * 10                        # Cabe no apto?
    s_dist = max(0, 1 - (lote.distancia_km / 1000)) * 10      # Até 1000km
    s_fam = familiaridade_score(lote) * 10                    # Conhece o produto?
    s_saz = sazonalidade_score(lote) * 10                     # Época certa?
    
    return s_roi + s_conf + s_liq + s_espaco + s_dist + s_fam + s_saz
```

```

  │
  ├── Categoria conhecida? (celular, perfume, informática)
  │     ├── ROI >= 50% e preço <= R$ 5K? → 🥇 BALA DE PRATA
  │     ├── ROI >= 30%? → 🟡 Analisar mais
  │     └── ROI < 30%? → 🔴 Passar

## 2.4 Estratégia de Lotes para os Primeiros 3 Meses

### Primeira compra (Mês 1)
**Perfil ideal:** Ticket baixo (R$ 2-5K), categoria conhecida, descrição detalhada

| Alvo | Por quê |
|------|---------|
| Celulares (iPhone, Xiaomi) | Alta demanda, fácil de testar, cabe no bolso |
| Perfumes importados | Leve, fácil de revender, margem boa |
| Brinquedos lacrados | Zero risco de defeito, fácil de vender |
| Informática (SSD, RAM) | Fácil de testar, demanda constante |

**Evitar na primeira compra:**
- ❌ Veículos (precisa de expertise mecânica + espaço)
- ❌ Lotes "DIVERSOS" (não sabe o que vem)
- ❌ Eletrodomésticos volumosos (não cabe em apto)
- ❌ Lotes acima de R$ 8K (não arriscar tudo de uma vez)

### Segunda compra (Mês 2-3)
**Perfil:** Médio (R$ 5-10K), se a primeira vendeu bem

| Alvo | Estratégia |
|------|-----------|
| Lote maior de celulares | Se provou que vende rápido, escalar |
| Veículo popular | Se tem garagem ou vende rápido |
| Moto popular | Mayor margem, mas precisa de espaço |

def preco_venda_ideal(custo_unitario: float, plataforma: str) -> float:
    """
    Calcula o preço de venda necessário para atingir margem líquida de 20%.

    Fórmula: Preço = Custo / (1 - TaxaEfetiva - MargemDesejada)

        taxa_efetiva = 0.13
        frete = 25.0  # frete médio PAC

        frete = 0.0

        taxa_efetiva = 0.13

    margem_desejada = 0.20  # 20%
    
    preco_minimo = (custo_unitario + frete) / (1 - taxa_efetiva - margem_desejada)

## 5.6 Política de Devolução (ML/Shopee)

```

  → Comprador abre reclamação no ML
  → ML avalia: se der razão ao comprador, devolução forçada
  → Se você descreveu o estado real: risco baixo

```

```
