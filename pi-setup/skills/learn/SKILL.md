# Learn Workflow

## Workflow

1. Capture what happened, why it matters, and the correct future behavior.
2. Categorize:
   - Team/process convention → `memory/team-conventions.md`
   - Recurring architecture/workflow pattern → `memory/project-patterns.md`
   - Bug/fix/session lesson → `memory/learning-log.md`
3. Append with date and concise context.
4. If inside workspace git repo, commit the memory update with `docs(memory): <topic>` after confirmation and branch safety checks.

## Threshold

Only save learnings that would help a future agent avoid a real mistake or recover important context.
