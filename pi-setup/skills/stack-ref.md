# Skill: /stack-ref

## Trigger
`how do I`, `what's the pattern for`, `show me the`, `reference for`, stack-specific questions.

## Workflow

1. **Identify stack** — ask or infer from current directory and files.
2. **Route to reference** — load the appropriate deep reference from `stack-ref/<topic>.md` if it exists.
3. **Answer concisely** — give the pattern, anti-pattern, and a minimal example.

## Available References
Create files in `stack-ref/` for your stack:
- `typescript.md`, `react.md`, `nextjs.md`, `python.md`, `go.md`, `rust.md`
- `postgres.md`, `redis.md`, `docker.md`, `aws.md`
- `nix.md`, `pulumi.md`, `terraform.md`

If a reference doesn't exist, answer from general knowledge and ask if one should be created.
