# Markdown Cheatsheet

Quick reference for Obsidian-flavored Markdown used in the workspace.

## Headings

```markdown
# H1
## H2
### H3
```

## Links

```markdown
[text](url)
[[wikilink]]          # Obsidian internal link
[[file#heading]]      # Link to heading
```

## Task Lists

```markdown
- [ ] Todo
- [x] Done
```

## Code

```markdown
`inline`
```language
block
```
```

## Tables

```markdown
| Col1 | Col2 |
|------|------|
| A    | B    |
```

## Callouts

```markdown
> [!note]
> Note content

> [!warning]
> Warning content

> [!tip]
> Tip content
```

## Frontmatter

```markdown
---
key: value
---
```

## Obsidian Specific

```markdown
#tag
^block-ref
![[embed]]
```
